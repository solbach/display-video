# Display Video
This small program shows the input video of an detached webcam

## usage
    $ ./displayVideo

## requires
* cmake 2.8
* openCV
* gcc

## tested on
* Ubuntu 14.04
* gcc 4.8.2
* cmake 2.8.12.2
* openCV 2.4.8
